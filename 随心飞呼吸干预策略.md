# 随心飞呼吸干预策略

## 1. 目标

本文用于统一随心飞场景中的呼吸干预频次、触发规则与文案策略，便于产品、算法、客户端协同对齐。

补充说明：

- 当前仓库阶段性用于手环 demo 调试与 Web 效果演示
- 其中 `fly_demo/` 为随心飞效果演示用的 Web demo
- 待手环 demo 调试完成后，本目录将继续用于随心飞安卓版本开发

本版结论如下：

- 呼吸干预全程最多执行 `2 次`
- 第 `1 次` 呼吸干预使用 `B 版`
- 第 `2 次` 呼吸干预使用 `A 版`
- 生理数据可按 `每 5 分钟` 采样，但不是每次采样都触发呼吸训练

## 2. 频次控制原则

### 2.1 基本判断

- 正常一次从监测数据到完成呼吸引导与训练，至少需要预留 `3 分钟`
- 如果干预过于频繁，用户会产生被反复打断的感受
- 呼吸训练更适合作为状态纠偏动作，而不是高频重复动作

因此，采样频次与呼吸触发频次应分开控制：

- `采样频次`：用于持续观察状态，可按每 `5 分钟` 执行一次
- `呼吸触发频次`：用于实际干预，需要设置次数上限和最小间隔

### 2.2 呼吸干预次数建议

| 随心飞时长 | 呼吸干预次数 |
| --- | --- |
| 2 分钟 | 0 次 |
| 3 ~ 12 分钟 | 最多 1 次 |
| 13 ~ 30 分钟 | 最多 2 次 |

### 2.3 呼吸干预触发节奏

- 第 1 次呼吸干预：在前段识别到异常状态后触发
- 第 2 次呼吸干预：仅在后续监测仍异常时触发
- 两次呼吸干预开始时间建议至少间隔 `10 分钟`

建议按以下方式理解：

- `0 / 5 / 10 / 15 / 20 / 25` 分钟可作为生理数据采样点
- 但真正进入呼吸干预的机会主要控制在前半段和后半段各 `1 次`
- 若第一次干预后状态明显改善，则不再进入第二次干预

## 3. 触发判定建议

### 3.1 日间场景

当识别到以下情绪时，可进入呼吸干预判断：

- 疲倦
- 兴奋
- 紧张
- 焦虑
- 沮丧

当识别到以下情绪时，不进入呼吸干预：

- 平静
- 满足
- 高兴

### 3.2 睡前场景

当识别到以下情绪时，可进入呼吸干预判断：

- 兴奋
- 紧张
- 焦虑
- 沮丧

说明：

- 疲倦通常有利于入睡，不建议在睡前场景中继续触发额外呼吸干预

### 3.3 第二次呼吸干预的触发条件

建议同时满足以下条件，再进入第二次呼吸干预：

- 第一次呼吸干预已完成
- 与第一次干预相比，后续监测结果仍处于异常状态
- 未回到平静、满足或高兴状态
- 距离第一次呼吸干预开始时间已满 `10 分钟`

## 4. 文案策略

## 4.1 总体策略

- 第 1 次呼吸干预使用 `B 版`
- 第 2 次呼吸干预使用 `A 版`

这样设计的原因：

- 第一次识别时，状态判断更适合用大类表达，文案更轻
- 第二次仍然异常时，说明状态持续存在，适合给出更具体的情绪引导

## 4.2 第一次呼吸干预：B 版

### 4.2.1 使用原则

- B 版采用大类归并表达
- 同一大类下可保留 2 到 3 条近义版本做随机，但不建议跨组混用
- B 版文案避免过度具体，重点是降低用户被“定义”的感觉

### 4.2.2 情绪归类建议

| 情绪 | 归类 |
| --- | --- |
| 兴奋 | 兴奋组 |
| 紧张 | 紧张 / 焦虑组 |
| 焦虑 | 紧张 / 焦虑组 |
| 疲倦 | 疲倦 / 沮丧组 |
| 沮丧 | 疲倦 / 沮丧组 |

### 4.2.3 B 版文案

| 情绪归类 | 页面提示语 | 语音引导 |
| --- | --- | --- |
| 兴奋 | 你现在似乎有些清醒，先做一组呼吸，慢慢缓下来 | 根据你当前的状态，接下来我们一起进行呼吸练习，慢慢把节奏放缓，让身体一点点安静下来。 |
| 紧张 / 焦虑 | 你现在似乎有些紧绷，先做一组呼吸，慢慢放松下来 | 根据你当前的状态，接下来我们一起进行呼吸练习，把注意力带回呼吸，让身体和心情一点点松下来。 |
| 疲倦 / 沮丧 | 你现在似乎有些低能量，先做一组呼吸，慢慢安顿下来 | 根据你当前的状态，接下来我们一起进行呼吸练习，先缓一缓，把呼吸放平，让自己慢慢稳定下来。 |

## 4.3 第二次呼吸干预：A 版

### 4.3.1 日间场景

| 情绪 | 页面提示语 | 语音引导 |
| --- | --- | --- |

| 兴奋 | 你似乎有些亢奋，来一起慢慢平静下来 | 此刻，你的心绪有些活跃。接下来，跟随均等呼吸，让呼吸慢下来，也让想法慢慢安静下来。 |
| 紧张 | 你似乎有些紧绷，来一起松一松 | 此刻，你的身体有些紧绷。接下来，跟随均等呼吸，一点一点松开身体，回到更放松的状态。 |
| 焦虑 | 你似乎有些不安，来一起缓一缓 | 此刻，你的内心有些不安。接下来，跟随均等呼吸，把注意力带回呼吸，陪自己慢慢安定下来。 |
| 沮丧 | 你似乎有些低落，来一起安顿一下心情 | 此刻，你的心情有些低落。接下来，跟随均等呼吸，轻轻托住这份感受，让自己缓一缓。 |

### 4.3.2 睡前场景

睡前场景统一使用 `478 呼吸`。

| 情绪 | 页面提示语 | 语音引导 |
| --- | --- | --- |
| 兴奋 | 你似乎还有些清醒，来一起慢慢安静下来 | 此刻，你的心绪还有些活跃。接下来，跟随 478 呼吸，让呼吸慢下来，让身体和想法一起安静下来，慢慢靠近睡眠。 |
| 紧张 | 你似乎有些紧绷，来一起放松入眠 | 此刻，你的身体还有些紧绷。接下来，跟随 478 呼吸，慢慢松开身体，把白天的用力一点点放下，准备进入睡眠。 |
| 焦虑 | 你似乎有些不安，来一起慢慢放下 | 此刻，你的内心还有些不安。接下来，跟随 478 呼吸，把注意力交给呼吸，让这份不安慢慢变轻，慢慢进入睡眠。 |
| 沮丧 | 你似乎有些沉重，来一起轻轻安顿下来 | 此刻，你的心情有些沉重。接下来，跟随 478 呼吸，先把这些感受轻轻放在一旁，让自己慢慢安静下来，进入睡眠。 |

## 5. 推荐流程

### 5.1 第一次呼吸干预

1. 用户进入随心飞
2. 按监测规则采集生理数据并识别状态
3. 若识别到需要干预的异常状态，则触发第 1 次呼吸干预
4. 第 1 次呼吸干预使用 B 版文案

### 5.2 第二次呼吸干预

1. 第 1 次呼吸干预结束后，继续进行后续监测
2. 若状态改善，则不再触发第二次呼吸干预
3. 若状态持续异常，且已满足最小间隔时间，则触发第 2 次呼吸干预
4. 第 2 次呼吸干预使用 A 版文案

## 6. 最终建议

本阶段建议先按以下规则推进：

- 随心飞全程最多执行 `2 次` 呼吸干预
- 第 1 次呼吸干预使用 `B 版`，采用大类表达
- 第 2 次呼吸干预使用 `A 版`，采用具体情绪表达
- 生理数据继续按 `每 5 分钟` 采样
- 呼吸干预开始时间之间建议至少间隔 `10 分钟`

一句话总结：

`第一次先粗分、轻干预；第二次再细分、强引导。`
